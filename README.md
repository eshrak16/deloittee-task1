# Deloitte Technology Virtual Internship 2026 – Task 1

This repository contains my solution for Task 1 of the Deloitte Technology Virtual Internship 2026.

The objective of this task was to create a Python-based solution that converts two different telemetry data formats into a single standardized structure. The implementation includes data transformation, timestamp conversion, and validation using the provided test cases.

As someone who is interested in Python , I found this task to be a valuable hands-on experience. It allowed me to work with JSON data, understand how different data models can be unified, and apply problem-solving skills in a practical business scenario.

All required test cases were completed successfully, and the final output matches the expected format provided in the task resources.

Thank you Deloitte and Forage for providing this learning opportunity.
